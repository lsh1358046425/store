## 项目介绍
Spring课程的课设，基于Spring && Spring MVC，使用阿里druid连接池、Mysql数据库、mybatis作对象映射(用的技术不多，看一眼POM便知)...前端使用Layui配合Jsp(很奇葩)。  
包含前台和后台管理，只适合初学者或者应付课设者作参考，代码写得烂敬请谅解。  
根目录附有数据库sql文件。
